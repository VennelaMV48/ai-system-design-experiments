# ✈️ AI Business Optimization for Air New Zealand

An AI-driven decision support framework designed to improve airline operational planning and capacity management.

This project explores how machine learning and AI systems can support large-scale operational decisions in the aviation industry.



## Problem

Airline operations require complex planning across multiple interconnected systems:

- flight scheduling
- network capacity planning
- commercial capacity allocation
- operational disruptions

Manual planning can lead to inefficiencies and suboptimal resource allocation.



## Proposed Solution

This project proposes an AI-driven planning system capable of analysing operational data and supporting decision making.

Key objectives:

- improve operational efficiency
- support data-driven planning decisions
- reduce manual analysis workload
